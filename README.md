# Match3
Match-3 style game built in Love2d

**This game is a work in progress. As of 9/6/2024, the game features include:**
- Menu Navigation
- Match-3 Game Play
- Indication of Hovered Cells
- Cascading Cell Refill
- Score with Combo Multipliers
